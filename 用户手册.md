## 1. 简介

**WeChat EXP** 是一个便携式微信聊天记录导出分析工具。无需安装 Python 或任何依赖库，只需一个 `wechat_exp.exe` 文件，复制到任何 Windows 电脑上即可使用。

**主要功能：**

| 功能 | 说明 |
|------|------|
| 密钥提取 | 自动从微信进程内存中提取数据库解密密钥 |
| 数据库解密 | 批量解密所有微信本地数据库 |
| 聊天导出 | 按人名、时间、关键词导出为 TXT 文件 |
| 员工批量导出 | 读取 Excel 员工表，智能匹配后批量导出 |
| 词云分析 | 中文分词 + 智能去噪，生成交互式 HTML 词云 |
| 综合报告 | 消息类型/时间分布/活跃会话等多维度分析报告 |

## 2. 准备工作

### 2.1 获取程序

从同事处复制 `wechat_exp.exe` 到你的电脑任意目录（如桌面或 D 盘根目录）。

> 程序为单文件，约 180~220MB，无需安装，双击即用。

### 2.2 启动微信

**重要：使用前必须先登录微信！**

程序需要从正在运行的微信进程中提取密钥，因此微信必须保持登录状态。

双击桌面上的微信图标，扫码登录后即可进行下一步。

### 2.3 双击运行

找到 `wechat_exp.exe`，双击运行。程序会自动打开一个命令行窗口，显示主菜单：

```

║                                          ║

```

```
  联系人: [全部/a] 或输入姓名关键词 > 张三
  时间范围: 最近 [30] 天 > 7
  关键词过滤: [无] > 报价
  输出目录: [exp/] >

  ✓ 导出完成: 1 人 / 23 条消息
  输出目录: exp/
```

各选项说明：

| 选项 | 说明 | 示例 |
|------|------|------|
| 联系人 | 输入 `a` 或直接回车 = 全部联系人；输入关键词 = 只导出名字含该词的联系人 | `张三`、`工作群` |

| 关键词过滤 | 直接回车 = 不过滤；输入文字 = 只保留含该词的聊天记录 | `报价`、`会议` |
| 输出目录 | 直接回车 = 默认 `exp/` 目录；也可输入自定义路径 | `E:\备份\`、`C:\Users\Admin\Desktop\chat\` |

## 5. 命令行模式

除了交互菜单，程序也支持命令行参数，适合定时任务或脚本调用。

### 基本用法

按 `Win + R`，输入 `cmd` 打开命令行，进入程序所在目录后执行：

```batch
REM 一键全流程（提取密钥 + 解密 + 导出最近30天）
wechat_exp.exe --auto

REM 仅提取密钥 + 解密
wechat_exp.exe --keys --decrypt

REM 导出最近7天、联系人含"张三"的记录
wechat_exp.exe --export --name=张三 --days=7

REM 导出含"报价"关键字的记录
wechat_exp.exe --export --keyword=报价 --days=30

REM 指定自定义输出目录
wechat_exp.exe --export --output=C:\Users\Admin\Desktop\chat_export
```

### 参数速查表

| 参数 | 说明 | 示例 |
|------|------|------|
| `--auto` | 一键全流程 | `--auto` |
| `--keys` | 仅提取密钥 | `--keys` |
| `--decrypt` | 仅解密 | `--decrypt` |
| `--export` | 导出聊天 | `--export` |
| `--name NAME` | 按姓名过滤 | `--name=张三` |
| `--days N` | 最近 N 天 | `--days=7` |
| `--keyword KW` | 关键词过滤 | `--keyword=报价` |
| `--db-dir PATH` | 手动指定微信数据目录 | `--db-dir=D:\xwechat_files\...` |
| `--output PATH` | 自定义输出目录 | `--output=E:\export\` |

### Windows 定时任务示例

1. 打开"任务计划程序"（Win+R → `taskschd.msc`）
2. 创建基本任务 → 触发器设为"每周一 9:00"
3. 操作 → 程序填写 `D:\wechat_exp\wechat_exp.exe`，参数填写 `--export --days=7 --output=E:\weekly\`
4. 完成。每周一自动导出最近一周聊天记录。

## 6. 输出文件说明

程序输出分为两部分：聊天导出在 `exp/` 目录，其他输出在 `output/` 目录：

```
wechat_exp/
├── exp/                       # 聊天记录导出（TXT格式）
│   ├── 张三.txt
│   ├── 李四.txt
│   └── 工作群1.txt
└── output/
    ├── all_keys.json          # 提取的密钥（加密存储，勿外传）
    ├── decrypted/             # 解密后的数据库（中间产物，可删除）
    │   ├── contact/
    │   │   └── contact.db
    │   ├── message/
    │   │   ├── message_0.db
    │   │   └── message_1.db
    │   └── session/
    │       └── session.db
    ├── employee_export/       # 员工批量导出（TXT格式）
    │   ├── 张三.txt
    │   └── 李四.txt
    ├── wordcloud/             # 词云分析（HTML格式，浏览器打开）
    │   ├── 全局词云.html
    │   └── 工作群1.html
    └── report.html            # 综合报告（HTML格式，浏览器打开）
```

| 文件类型 | 用途 | 打开方式 |
|----------|------|----------|
| `.txt` | 聊天记录文本 | 记事本、Notepad++、VS Code 等任意文本编辑器 |
| `.html` | 词云 / 分析报告 | 浏览器（Chrome / Edge / Firefox） |

> **安全提醒**：`output/all_keys.json` 包含数据库解密密钥，**切勿外传**！

## 7. 常见问题

### Q1：双击后窗口一闪而过？

**原因**：微信没有启动。

**解决**：先启动微信并扫码登录，再双击 `wechat_exp.exe`。

---

### Q2：提示"Weixin.exe 未运行"？

```
检测微信... Weixin.exe 未运行!

请先启动微信并登录，然后重试。
按 Enter 返回主菜单...
```

**原因**：同上，微信进程未运行或未登录。

**解决**：确保微信已登录（而非仅停留在登录界面）。

---

### Q3：提示"未自动检测到"微信数据目录？

```
搜索微信数据目录... 未自动检测到!

  请按以下步骤手动查找:
  1. 打开微信，点击左下角「三」→「设置」
  2. 选择「文件管理」→ 查看「文件目录」
  3. 进入该目录，找到 xwechat_files\<微信号>\db_storage

  请粘贴 db_storage 完整路径:
```

**原因**：微信安装在非标准位置，或使用了便携版微信。

**解决**：按提示步骤找到路径，复制后粘贴到窗口中回车即可。路径通常类似：
- `D:\xwechat_files\wxid_xxxxxxxxxxxxxx\db_storage`
- `C:\Users\你的用户名\Documents\xwechat_files\wxid_xxx\db_storage`

---

### Q4：部分数据库提示"无密钥"？

```
SKIP: message/message_3.db (无密钥)
```

**原因**：该数据库的密钥尚未加载到微信内存中。微信只在需要时加载对应数据库的密钥。

**解决**：
1. 在微信中打开更多聊天窗口（特别是较久远的聊天记录）
2. 滚动浏览不同联系人的聊天记录
3. 重新运行密钥提取（单独选 `[2]` 提取密钥 + `[3]` 解密）

---

### Q5：员工匹配数量太少？

**原因**：微信联系人的备注名、昵称与 Excel 中的员工姓名不一致。

**解决**：
- 在微信中给同事设置备注为真实姓名（这是提高匹配率最有效的方法）
- 在菜单 `[5]` 中降低"最低匹配度"（如改为 20）
- 确保 Excel 中姓名列的格式规范（无多余空格、无特殊字符）

---

### Q6：词云中有大量无意义词汇？

如"好的""收到""哈哈哈"等。

**解决**：程序已内置三层智能过滤（~800 停用词 + 词性过滤 + 上下文降权），能过滤大多数无意义词。如仍有个别词汇干扰，可联系开发者添加到停用词表。

---

### Q7：生成的 HTML 文件打开是空白的？

**原因**：ECharts 图表库从 CDN 加载，需要网络连接。

**解决**：确保电脑可以访问互联网。首次打开可能需要几秒钟加载图表库。

---

### Q8：Python 版本要求？

**EXE 用户无需关心！** 直接双击 `wechat_exp.exe` 即可，不需要安装 Python。

只有开发者从源码运行时才需要 Python 3.8+，详见 `开发手册.md`。

---

### Q9：支持微信 3.x 吗？

不支持。微信 4.x 采用了新的加密方案（SQLCipher 4），3.x 使用不同的加密方式。

判断微信版本：点击左下角「三」→「设置」→「关于微信」查看版本号。

---

### Q10：输出目录在哪里？

聊天导出在 `wechat_exp.exe` **所在目录**下的 `exp/` 文件夹中。其他输出（密钥、报告、词云等）在 `output/` 文件夹中。

例如：
- EXE 放在 `D:\wechat_exp\wechat_exp.exe`
- 聊天导出在 `D:\wechat_exp\exp\`
- 报告等在 `D:\wechat_exp\output\`

> 如果找不到，在程序窗口中会显示完整的输出路径。

## 8. 进阶技巧

### 技巧 1：定期备份聊天记录

将 `wechat_exp.exe` 放在固定目录，使用 Windows 任务计划程序，每周自动导出：

```
wechat_exp.exe --auto --output=E:\备份\微信记录\
```

### 技巧 2：针对性导出

如果只想导出某个项目的讨论记录：

1. 先在微信中为该联系人或群聊设置好备注名
2. 运行程序选择 `[4]`
3. 输入备注名关键词 + 相关时间范围 + 项目关键词（如"需求""排期"）
4. 得到精准的相关聊天记录

### 技巧 3：快速定位输出文件

程序运行完成后会显示完整输出路径，如：

```
报告已生成: D:\wechat_exp\output\report.html
```

在文件资源管理器的地址栏中直接粘贴这个路径即可快速打开。

### 技巧 4：清理解密数据

`output/decrypted/` 目录存放解密后的数据库文件，其中包含你的聊天记录。如不再需要可手动删除以释放磁盘空间（通常几百 MB）。

下次分析时会自动重新解密，不留痕迹。

### 技巧 5：多人共用同一电脑

如果电脑上有多个微信账号，程序会自动选择最近活跃的账号进行导出。如需切换账号，在新账号登录后重新运行程序即可。
